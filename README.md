# Static Color Depth Visualizer

## Local use
Open `index.html` in a browser.

If your browser blocks local image loading from `file://`, run a tiny local server in this folder:

```bash
python3 -m http.server 8080
```

Then open `http://127.0.0.1:8080`.

## GitHub Pages
Upload this folder to a GitHub repo (or repo subfolder), enable Pages, and use `index.html` as the entry page.
